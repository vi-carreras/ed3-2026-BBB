# ed3-2026-BBB

Repositorio para trabajo práctico final de la materia.

Juego de reacción competitivo para 2 jugadores sobre **LPC1769** — medición de tiempos de reacción con precisión de microsegundos usando Timer Input Capture, generación de audio por DAC + DMA, LCD 16x2 por I2C, y comunicación serial con app Java.

- [Especificación del sistema](Juego_de_reaccion.md)
- `src/` — firmware principal (`main.c`) + librería (`general.c`)
- `inc/` — headers de la librería (`general.h`)
